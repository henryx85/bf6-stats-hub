## Configuration

No separate configuration files are required. Preferences are kept automatically between sessions.

## FAQ

**How do I update to a new version?**  
Grab the newest release from the [releases page](https://github.com/henryx85/bf6-stats-hub/releases) and swap out your current files. Your saved settings will stay in place.

**Can I use this on mobile devices?**  
Yes. The interface is fully responsive and works on smartphones and tablets in a mobile browser.

**Why aren't my searches saving?**  
Make sure browser local storage is allowed. Privacy settings or extensions can interfere with it.

**How do I clear my search history?**  
Open the settings menu and choose "Clear Recent Searches." This removes all stored player lookups.

**Is there a desktop version?**  
No, this project is web-only. You can still bookmark it or save it as a progressive web app on supported browsers.
